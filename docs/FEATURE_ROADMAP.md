# HabitaPlot Enhanced Features Roadmap

## MVP Phase (Week 1-2)
### 1. Offline-First Mobile Support (PWA)
- Service Worker with cache-first strategy
- Sync transactions when online
- Offline listing browse
- Status: **In Progress**

### 2. QR Code POS & Scanning
- Generate QR codes for listings
- `jsQR` library for scanning
- Quick checkout flow
- Status: **In Progress**

## Phase 2 (Week 3-4)
### 3. WhatsApp Business Integration
- Order notifications via WhatsApp
- Customer engagement
- Integration: Official API
- Status: **Planned**

### 4. Multi-Currency & Tax Configuration
- Support UGX, USD, KES, ZAR
- Tax rate management per branch
- Automatic tax calculation
- Status: **Planned**

### 5. AI Demand Forecasting
- Time-series forecasting (Prophet or LSTM)
- Inventory alerts
- Seasonal trend analysis
- Status: **Planned**

### 6. Analytics Dashboard
- Sales by provider/period
- Top listings performance
- Revenue trends
- Export: CSV, PDF, JSON
- Status: **Planned**

### 7. Business Branch Management
- Multi-branch support
- Per-branch inventory
- Branch-level reporting
- Status: **Planned**

## Implementation Priority
1. ✅ PWA + offline sync
2. ✅ QR barcode scanning
3. 📅 WhatsApp Business API
4. 📅 Multi-currency & tax
5. 📅 AI forecasting (Prophet)
6. 📅 Advanced analytics
